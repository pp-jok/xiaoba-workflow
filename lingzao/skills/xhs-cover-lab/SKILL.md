# Xiaohongshu Cover Lab

Do not generate a Lingzao / XHS cover from one vague sentence if the user wants
a good result. First ask for reference and style constraints.

Search phrases: lingzao xhs cover lab, Xiaohongshu cover design, XHS cover
analysis, 小红书封面诊断, 小红书封面文案.

## Ask First

If missing, ask briefly:

- 你有没有参考图？
- 有没有想要的配色？
- 是小红书封面、图文内页、公众号封面，还是产品海报？

## Cover Routes

Common routes:

- face-led keyword cover
- no-person knowledge card
- interaction prompt cover
- text-dense screenshot-style graphic note
- local life / food / travel visual cover
- product or ecommerce conversion cover
- WeChat article cover

## Output

- recommended cover type
- cover keyword
- cover copy
- visual hierarchy
- reference-image notes
- image-generation brief, if needed
- what to avoid
