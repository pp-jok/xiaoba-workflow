# Xiaohongshu Single Note Breakdown

This lightweight Lingzao sub-skill helps users learn from one Xiaohongshu / XHS
note without copying it.

Search phrases: lingzao xhs note breakdown, Xiaohongshu viral note analysis,
XHS content breakdown, 小红书爆款笔记拆解, 小红书单条笔记分析.

## Inputs

- note link, screenshots, or pasted copy
- visible metrics if available: likes, saves, comments
- cover image, title, caption, comment screenshots, or video transcript
- user's own track and format: graphic note, spoken video, or Vlog

## Workflow

1. Surface layer: title, cover, visible promise, keyword, first impression.
2. Structure layer: hook, outline, turns, proof, ending, CTA.
3. Format layer: graphic-note pages, spoken script, Vlog storyboard, or
   shooting/editing rhythm when visible.
4. Comment-demand layer: what users are asking, saving, debating, or doubting.
5. Viral mechanism: why this content worked for this creator and audience.
6. Learnability test: what an ordinary user can learn versus what is hard to
   copy.
7. Adaptation: turn it into the user's own topic, format, and next draft.

## Output

- Why it may have worked
- What can be learned
- What should not be copied
- Adapted title, cover direction, outline, and draft direction
- one next action for the user
