# Benchmark Copy Rewrite

Do not directly imitate the benchmark copy. First separate the Lingzao / XHS
template from the original creator's content.

Search phrases: lingzao benchmark copy rewrite, xhs copywriting template,
Xiaohongshu caption rewrite, 对标文案仿写, 小红书文案模板.

## Inputs

- 1-3 benchmark copies, captions, scripts, or screenshots
- user's topic, product, audience, account direction, or customer pain
- desired platform: Xiaohongshu graphic note, spoken video, Moments, WeChat,
  or another format

## Workflow

1. Extract structure: hook, middle route, proof, turn, ending, CTA.
2. Extract style: tone, sentence length, emotion level, question rhythm,
   spoken feel, or dry-good density.
3. Build slots: audience, pain, scene, product, city, personal story, proof,
   method, result, and next action.
4. Mark non-copyable parts: exact sentences, life experience, private data,
   creator identity, fake numbers, exaggerated claims, and comments.
5. Fill the user's truthful material into the slots.
6. Run quality gate: not copied, not empty, not off-track, not repetitive, and
   publishable.

## Output

- template skeleton
- style parameters
- slot table
- non-copyable list
- 1-3 adapted drafts
- short self-check explaining where it is similar and where it is original
